# Summer Game Fest Bingo 2026

Одностраничный интерактивный лендинг-бинго для Twitch-стримера **HappyZerg**, сделанный специально для презентации **Summer Game Fest 2026**.

Проект позволяет зрителям отмечать события и предсказания во время шоу: анонсы игр, трейлеры, фразы вроде `World Premiere`, неожиданных гостей и другие типичные моменты летних игровых презентаций. При сборе линии BINGO на экране появляется праздничная анимация.

## Возможности

- Интерактивное bingo-поле 5x5.
- Визуальное изменение карточек при клике.
- Автоматическое определение BINGO по строкам, колонкам и диагоналям.
- Таймер до старта Summer Game Fest 2026 по московскому времени.
- Счетчик для карточки `World Premiere 5+ раз`.
- Ссылки на Twitch и Telegram HappyZerg.
- Адаптивный неоновый дизайн в стиле Summer Game Fest.
- Деплой на GitHub Pages.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой на GitHub Pages
